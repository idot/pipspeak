# seqspec

The seqspec file was generated by an LLM and not validated

pipspeak checks for all barcodes between the spacers with 1 mismatch each and outputs a file with
corrected barcodes merged + UMI that can be fed to STAR


```bash
pipspeak \
        --loglevel info \
        --config $config \
        --prefix $prefix \
        --r1 ${r1} \
        --r2 ${r2} \
        --offset 8 \
        --umi-len 12 \
        --umi-offset 10 \
        --threads $task.cpus
```
